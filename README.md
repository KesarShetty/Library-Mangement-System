📚 Smart Library Management System
A modern and responsive Library Management System developed using HTML, CSS, and JavaScript. This project helps users manage books, issue and return books, search for books, and maintain library records dynamically.

✨ Features
🔐 User Login and Registration System
💾 Data stored using Local Storage
📖 Add New Books Dynamically
🔍 Search Books by Title or Author
📕 Issue and Return Books
🗑 Delete Books
📊 Dashboard with:
Total Books
Issued Books
Available Books
🌙 Dark Mode Toggle
🎨 Modern UI with Glassmorphism Design
⚡ Dynamic and Animated Interface
🛠 Technologies Used
HTML5
CSS3
JavaScript
Local Storage API
📂 Project Pages
1️⃣ Login Page
Users can log in using registered credentials.

2️⃣ Register Page
New users can create an account and store credentials locally.

3️⃣ Main Dashboard
Displays all library functionalities including:

Book Management
Search System
Dashboard Statistics
Book Issue & Return System
🚀 How to Run
Download or clone the project
Open index.html
Register a new account
Login and access the dashboard
